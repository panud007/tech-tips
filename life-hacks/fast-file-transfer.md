### 1. Nearby Share (Windows & Android) âš¡

Ini "AirDrop"-nya Android & Windows. Sekarang sudah built-in!

**Caranya:**

1. **Di PC**: Install [Nearby Share for Windows](https://www.android.com/better-together/nearby-share-app/) (resmi Google).
2. **Di HP**: Aktifkan Nearby Share di Quick Settings.
3. Klik Share di file -> Pilih Nearby Share -> Klik nama PC.
4. **Kecepatan**: Sangat Cepat (via WiFi Direct).

### 2. Snapdrop / PairDrop (Universal) ğŸŒ

Gak perlu install aplikasi. Cukup buka browser.
Bisa: Android, iPhone, Windows, Mac, Linux.

**Caranya:**

1. Buka [snapdrop.net](https://snapdrop.net) di **KEDUA** device.
2. Pastikan satu jaringan WiFi.
3. Nama device lain bakal muncul kayak radar.
4. Tap nama device -> Pilih file -> Kirim.
