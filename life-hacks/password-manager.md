Aplikasi ini tugasnya mengingat password rumit buat kamu. Kamu cuma perlu ingat **Satu Master Password**.

### Rekomendasi: Bitwarden ğŸ›¡ï¸

Gratis, Aman, Open Source.

**Cara Setup:**

1. **Download Bitwarden** di HP dan Extension di Browser (Chrome/Edge).
2. **Buat Akun**. Masukkan Master Password yang *Kuat tapi Mudah Diingat*.
   - Contoh Frasa: `KucingMerahMakanSoto2024!` (Panjang + Angka + Simbol).
3. **Mulai Simpan**: Tiap login web, Bitwarden bakal tanya "Save Password?". Klik Yes.
4. **Generate Password**: Pas daftar akun baru, suruh Bitwarden bikin password acak.
   - Contoh: `Xy7#mP9$vL2@kR` (Gak mungkin ditebak hacker).

### Kenapa Gak Pakai Chrome Password Manager?

Chrome Password Manager oke, tapi kalau PC kamu dipinjam orang dan Chrome dibuka, mereka bisa lihat semua password kamu kalau PC gak dikunci. Bitwarden bisa set auto-lock.
