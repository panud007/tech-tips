## âœ… Solusi: Windows Search Tricks

Kalau gak boleh install aplikasi, pakai operator ini di search box Explorer:

- `kind:image` -> Cari gambar aja.
- `date:today` -> Cari file yang dibuat hari ini.
- `size:giant` -> Cari file >128MB.
- `namafile content:rahasia` -> Cari file bernama "namafile" DAN di dalamnya ada tulisan "rahasia".
